# VELOCITY OFFROAD

A modern racing team website showcasing our passion for offroad racing and engineering excellence.

## Live Website
🌐 **[velocityoffroad.netlify.app](https://velocityoffroad.netlify.app/)**

## About
VELOCITY OFFROAD is a college racing team dedicated to designing, building, and racing high-performance offroad vehicles. Our website features team information, project showcases, event updates, and sponsor partnerships.

## Technology
Built with React, TypeScript, and Tailwind CSS for a fast, responsive user experience.

## Quick Start
```bash
git clone https://github.com/deepak-158/Velocity-offroad.git
cd Velocity-offroad/frontend
npm install
npm start
```

Visit `http://localhost:3000` to view the application.
